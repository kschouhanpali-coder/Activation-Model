Activation Functions are mathematical functions used in neural networks to introduce non-linearity and help AI models learn complex patterns. They decide which neurons should activate, improving prediction accuracy in tasks like image recognition, chatbots, NLP, and deep learning systems.

## 📌 Project Overview

**Neural Genesis** is a dark-themed, interactive learning platform designed to visualize and explain core concepts of neural networks and deep learning. It features live simulations and data-rich visualizations — all accessible through a sleek **Command Hub** sidebar.

---

## 🚀 Features

### 🛰️ Mission Control
- Central landing page with "Activation Visualizer" hero section
- Full-brain neural imagery and strategic intel cards
- Overview of non-linear edge concepts (ReLU, Swish, Sigmoid, Softmax)

### 🧠 Neural Archive Dive
- AI-powered knowledge explorer for deep neural network datasets
- Advantages, Disadvantages, and Use Cases breakdown
- Integrated **Select Function** dropdown with Sigmoid graph viewer
- Use cases: Medical Imaging, Self-Driving Tech, Satellite Tracking, NLP Research

### 🧪 Live Synaptic Lab
- Real-time neuron simulator with interactive sliders:
  - **Input (x)**, **Weight (w)**, **Bias (b)**
- Live **activation function switcher** (Sigmoid, ReLU, Tanh, etc.)
- Real-time output display (e.g., `0.8808`)
- Live graph rendering with Plotly-style charts

### 🗺️ Neural Map — Data Architecture Flow
- Explains structured data pipelines and AI/ML system architecture
- **Synaptic Pulse Visualization** — interactive scatter/network graph
- Covers cloud computing, big data, and AI data flow concepts

### 📊 Matrix View — Topology Matrix
- Side-by-side comparison cards for 6 activation functions:
  | Function | Range | Use Case |
  |---|---|---|
  | Sigmoid | (0, 1) | Binary Classification |
  | ReLU | [0, ∞) | CNNs |
  | Tanh | (-1, 1) | RNNs |
  | Leaky ReLU | (-∞, ∞) | GANs |
  | Softmax | [0, 1] | Object Detection |
  | Swish | (-0.278, ∞) | EfficientNets |
- Mathematical formulas rendered for each function

### 📉 Gradient Flow Analytics
- Visual explainer for backpropagation and gradient movement
- High-quality AI-generated diagram: Layer 1 → Layer 2 → Layer 3
- Explains: Vanishing Gradient vs ReLU Solution
- Comparison panel with Sigmoid formula vs ReLU formula
